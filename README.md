# 빅데이터 분석기사 실기 모의고사

도서 구매자 전용 학습 서비스 — Python·R 양 트랙 운영.

## 구조 (예정)

- `/` — 트랙 인식 + 모바일/PC 분기
- `/?track=python` — Python(Colab) 트랙 진입
- `/?track=r` — R(Posit Cloud RStudio) 트랙 진입

## 기술 스택

HTML5 + CSS3 + Vanilla JS (프레임워크 없음). 정적 사이트, Vercel 자동 배포.

## 상태

🚧 구축 중 — 임시 랜딩 페이지만 배포되어 있으며, 도서 발간 일정에 맞춰 정상 오픈 예정.
